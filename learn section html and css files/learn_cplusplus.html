1. Introduction to C++

What is C++?

C++ is a versatile programming language that builds upon C by adding object-oriented programming capabilities. Created by Bjarne Stroustrup in 1979, C++ is widely used in software development, game development, and competitive programming.

Key Characteristics of C++

  • Object-Oriented: Supports classes, inheritance, and polymorphism.
  • Compiled Language: Known for high performance and extensive control over system resources.
  • Standard Library: Offers a rich set of libraries and templates.
