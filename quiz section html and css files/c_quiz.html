C Language Quiz with Explanations

1. What is the size of an integer in C?

This is incorrect. The size of an integer is typically 4 bytes in most systems.

This is correct! In most systems, an integer is 4 bytes.

2. Which function is used to read a string in C?

This is incorrect. The function gets() is commonly used to read a string until a newline.

This is correct! Note that fgets() is preferred over gets() for safety.

3. Which data type is used to store a single character in C?

This is incorrect. The char data type stores single characters.

This is correct! The char data type is used for single characters.

4. What is the output of the following code:


 printf("%d", 5 + 2 * 3);

This is incorrect. The multiplication operator has higher precedence, so the answer is 11.

This is correct! According to operator precedence rules, the answer is 11.

5. Which keyword is used to define a constant in C?

This is incorrect. The const keyword is used to define constants in C.

This is correct! The const keyword defines a constant in C.

6. Which symbol is used for comments in C?

This is incorrect. Comments in C use // for single-line and /* */ for multi-line.

This is correct! In C, // is used for single-line comments and /* */ for multi-line comments.

7. What will be the output of the following code?


 int a = 5;
 int b = a++;
 printf("%d %d", a, b);

This is incorrect. The variable 'b' gets the initial value of 'a' before the increment, so the answer is 6 5.

This is correct! The value of 'a' becomes 6 after incrementing, but 'b' holds the original value of 5.

8. Which of the following operators has the highest precedence in C?

This is incorrect. The parentheses operator () has the highest precedence in C.

This is correct! Parentheses have the highest precedence, allowing grouped expressions to be evaluated first.

9. Which of the following correctly declares a pointer to an integer?

This is incorrect. The correct syntax for declaring a pointer is 'int *ptr'.

This is correct! 'int *ptr' declares a pointer to an integer in C.

10. What will the following code output?


 int a = 3, b = 2;
 printf("%d", a > b ? a : b);

This is incorrect. The ternary operator returns the larger of 'a' or 'b', which is 3.

This is correct! Since 'a' is greater than 'b', the output is 3.

11. What is the purpose of the "return 0;" statement in the main() function?

This is incorrect. "return 0;" indicates that the program has terminated successfully.

This is correct! "return 0;" is a standard way to signal that the program finished without errors.

12. What will be the output of the following code?


 int x = 0;
 if (x)
    printf("True");
 else
    printf("False");

This is incorrect. In C, 0 is considered false, so the output is "False".

This is correct! Since x is 0, the condition evaluates to false, and "False" is printed.

13. What is the output of the following code?


 #include 
 void func(int *x) {
     *x = 10;
 }
 int main() {
     int a = 5;
     func(&a);
     printf("%d", a);
     return 0;
 }

This is incorrect. The value of 'a' is updated to 10 in the function, as we're passing the address of 'a' (using a pointer).

This is correct! The value of 'a' is modified by the function via the pointer, and the output will be 10.

14. What will be the output of the following code?


 #include 
 int main() {
     int a = 5;
     if (a = 0)
         printf("True");
     else
         printf("False");
     return 0;
 }

This is incorrect. The assignment 'a = 0' sets 'a' to 0, which evaluates as false.

This is correct! The assignment inside the if condition makes 'a' 0, which evaluates to false, so "False" is printed.

15. Which of the following is the correct way to declare a function pointer?

This is incorrect. The correct syntax for declaring a function pointer is 'int (*func)();'.

This is correct! 'int (*func)();' correctly declares a pointer to a function that returns an integer.

16. What is the output of the following code?


 #include 
 int main() {
     int i = 5;
     printf("%d", i++);
     return 0;
 }

This is incorrect. The 'i++' increments the value after printing it, so the value printed is 5.

This is correct! The post-increment operator prints the current value of 'i' (5) before incrementing it.

17. What is the output of the following code?


 #include 
 int main() {
     int a = 10;
     int b = 20;
     int c = a > b ? a : b;
     printf("%d", c);
     return 0;
 }

This is incorrect. The ternary operator chooses the larger value, which is 20.

This is correct! The ternary operator selects 'b', as it is greater than 'a'.

18. What is the output of the following C code:


 int a = 10;
 int b = 20;
 printf("%d", a + b > 15);
This is incorrect. The condition a + b > 15 evaluates to true (since 30 > 15), which is represented as 1 in C.
This is correct! The condition a + b > 15 evaluates to true, which is represented as 1 in C.
