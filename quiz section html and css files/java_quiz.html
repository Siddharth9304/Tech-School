Java Language Quiz with Explanations

1. What is the default value of an int variable in Java?

This is incorrect. The default value of an int in Java is 0.

This is correct! The default value of an int in Java is 0.

2. Which of the following is the correct syntax to declare a Java variable?

This is incorrect. The correct syntax for variable declaration in Java is all of the above.

This is correct! All of the above options are correct for declaring a variable in Java.

3. What is the output of the following code snippet? System.out.println(10 + 20 + "30");

This is incorrect. The correct output is 1030.

This is correct! The output is 1030 due to string concatenation.

4. Which of the following keywords is used to prevent inheritance in Java?

This is incorrect. The correct keyword to prevent inheritance in Java is 'final'.

This is correct! The 'final' keyword prevents inheritance in Java.

5. What will be the output of the following code snippet? System.out.println(5 + 3 + \"Hello\");

This is incorrect. The correct output is 8Hello due to string concatenation.

This is correct! The output is 8Hello because integers are added first, followed by string concatenation.

6. Which of the following is the correct way to declare a constant variable in Java?

This is incorrect. The correct syntax is 'final int num = 10;'.

This is correct! The correct way to declare a constant variable is 'final int num = 10;'.

7. What will be the output of the following code snippet? int[] arr = {1, 2, 3}; System.out.println(arr[3]);

This is incorrect. The correct answer is an ArrayIndexOutOfBoundsException since the array has only indices 0, 1, and 2.

This is correct! Attempting to access arr[3] will throw an ArrayIndexOutOfBoundsException.

8. What is the result of the expression 5 + 3 * 2 in Java?

This is incorrect. The correct result is 11 because multiplication has higher precedence than addition.

This is correct! The result is 11, as multiplication happens before addition.

9. Which of the following statements is true about Java?

This is incorrect. The correct statement is that Java is both compiled and interpreted.

This is correct! Java is first compiled into bytecode and then interpreted by the JVM.

10. What is the output of the following code snippet? String str = "Java"; str = str + " Programming"; System.out.println(str);

This is incorrect. The correct output is "Java Programming" due to the string concatenation.

This is correct! The output is "Java Programming" due to string concatenation.

11. What is the difference between the "==" operator and the ".equals()" method in Java?

This is incorrect. The correct difference is "==" checks reference equality, and ".equals()" checks object equality.

This is correct! "==" checks if two references point to the same object, while ".equals()" checks if the objects themselves are equal.

12. What does the "super" keyword refer to in Java?

This is incorrect. The "super" keyword refers to the immediate superclass of the current object.

This is correct! "super" refers to the immediate superclass of the current object.

13. What will be the output of the following code snippet? int x = 5; System.out.println(x++ + ++x);

This is incorrect. The correct output is 12 because x++ increments after the value is used and ++x increments before the value is used.

This is correct! The output is 12 because x++ evaluates to 5 and ++x increments x to 7, giving 5 + 7 = 12.

14. What is the result of the following code snippet? String s1 = "Hello"; String s2 = "Hello"; System.out.println(s1 == s2);

This is incorrect. The correct answer is true because string literals are interned in Java, meaning s1 and s2 reference the same object.

This is correct! String literals in Java are interned, so s1 and s2 refer to the same object.

15. Which of the following will cause a NullPointerException in Java?

This is incorrect. The correct cause is calling a method on a null object reference, which leads to a NullPointerException.

This is correct! Calling a method on a null object reference will throw a NullPointerException.

16. What is the output of the following code? ArrayList list = new ArrayList<>(Arrays.asList(1, 2, 3)); list.remove(Integer.valueOf(2)); System.out.println(list);

This is incorrect. The correct output is [1, 3] because remove(Integer.valueOf(2)) removes the element 2 from the list.

This is correct! The element 2 is removed, leaving the list [1, 3].

17. What will be the output of the following code? int x = 10; int y = 20; if(x > y) { System.out.println("x is greater"); } else { System.out.println("y is greater"); }

This is incorrect. The correct output is "y is greater" since 10 is less than 20.

This is correct! Since x is 10 and y is 20, "y is greater" is printed.

18. What is the output of the following code? ArrayList list = new ArrayList<>(Arrays.asList(1, 2, 3)); list.remove(Integer.valueOf(2)); System.out.println(list);

This is incorrect. The correct output is [1, 3] because remove(Integer.valueOf(2)) removes the element 2 from the list.

This is correct! The element 2 is removed, leaving the list [1, 3].
