C++ Language Quiz with Explanations

1. What is the output of the following code?


 #include 
 using namespace std;
 int main() {
     int x = 5;
     cout << x++;
     return 0;
 }

This is incorrect. The post-increment operator prints the current value of 'x' (5) before incrementing it.

This is correct! The post-increment operator prints 5 and then increments 'x'.

2. What will be the output of the following code?


 #include 
 using namespace std;
 int main() {
     bool x = false;
     cout << x;
     return 0;
 }

This is incorrect. In C++, the boolean value 'false' is represented as 0.

This is correct! 'false' is printed as 0 in C++.

3. What is the output of the following code?


 #include 
 using namespace std;
 int main() {
     int a = 5, b = 0;
     cout << a / b;
     return 0;
 }

This is incorrect. Dividing by zero results in undefined behavior in C++.

This is correct! Dividing by zero leads to undefined behavior in C++.

4. What will be the output of the following code?


 #include 
 using namespace std;
 int main() {
     int a = 5;
     if (a = 0)
         cout << "True";
     else
         cout << "False";
     return 0;
 }

This is incorrect. The assignment 'a = 0' makes 'a' 0, which evaluates to false.

This is correct! The assignment 'a = 0' results in false, so "False" is printed.

5. Which of the following is the correct way to declare a function pointer in C++?

This is incorrect. The correct syntax for declaring a function pointer is 'int (*func)();'.

This is correct! 'int (*func)();' correctly declares a pointer to a function that returns an integer.

6. What is the output of the following code?


 #include 
 using namespace std;
 int main() {
     int i = 5;
     cout ⩽⩽ i++;
     return 0;
 }

This is incorrect. The post-increment operator prints 5 and then increments 'i'.

This is correct! The post-increment operator prints 5 and then increments 'i' to 6.

7. What will be the output of the following code?


 #include 
 using namespace std;
 int main() {
     int a = 5;
     cout << ++a;
     return 0;
 }

This is incorrect. The pre-increment operator increments 'a' before printing it.

This is correct! The pre-increment operator increments 'a' to 6 before printing it.

8. Which of the following is a valid declaration of a constant in C++?

This is incorrect. The correct way to declare a constant in C++ is by using 'const' or 'const int'.

This is correct! 'const int x = 10;' correctly declares a constant integer.

9. What is the purpose of the 'virtual' keyword in C++?

This is incorrect. The 'virtual' keyword is used to enable runtime polymorphism in C++.

This is correct! The 'virtual' keyword is used to define virtual functions, allowing for dynamic dispatch and polymorphism.

10. What will be the output of the following C++ code?


 #include 
 using namespace std;
 int main() {
     int a = 10;
     int b = 3;
     cout << a / b;
     return 0;
 }

This is incorrect. The virtual function mechanism will invoke the derived class function even when called through a base class pointer.

This is correct! The virtual function in the derived class is called because of dynamic dispatch.

11. What is the size of a pointer in C++?

This is incorrect. The size of a pointer depends on the system architecture.

This is correct! The size of a pointer depends on the architecture (typically 4 bytes for 32-bit systems and 8 bytes for 64-bit systems).

12. What is the output of the following code?


 #include 
 using namespace std;
 int main() {
     int a = 10;
     int b = 5;
     cout << a % b;
     return 0;
 }

This is incorrect. The modulus operator returns the remainder of the division.

This is correct! 10 % 5 is 0 because the remainder when dividing 10 by 5 is 0.

13. What is a constructor in C++?

This is incorrect. Constructors are special functions used to initialize an object when it's created.

This is correct! A constructor is called automatically when an object is created to initialize it.

14. What is the result of dividing by zero in C++?

This is incorrect. Division by zero causes undefined behavior in C++.

This is correct! Division by zero results in undefined behavior in C++.

15. How do you declare a pointer in C++?

This is incorrect. The correct syntax for declaring a pointer is 'type *name'.

This is correct! 'int *ptr;' is the correct syntax for declaring a pointer to an integer.

16. What is the purpose of the 'new' operator in C++?

This is incorrect. The 'new' operator is used to allocate memory dynamically on the heap.

This is correct! The 'new' operator is used to allocate memory dynamically on the heap.

17. What is the output of the following code?


        #include 
        using namespace std;
        int main() {
            int a = 10;
            int b = 3;
            cout << a / b;
            return 0;
        }

This is incorrect. Since both 'a' and 'b' are integers, the division will result in an integer value.

This is correct! The division will result in 3 because it truncates the decimal part in integer division.

18. What will happen if a function does not return a value in C++?

This is incorrect. If a non-void function does not return a value, it leads to undefined behavior.

This is correct! A non-void function must return a value, and failing to do so results in undefined behavior.
